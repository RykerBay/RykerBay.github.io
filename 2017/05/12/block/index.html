block

Blocks（块）

定义

带有 局部变量 的 匿名 函数 (anonymous functions together with automatic (local) variables)

语法

^return_type (parameters) { ...}

return_type和parameters可省略

Block类型变量

定义

Block objects are a C-level syntactic and runtime feature. They are similar to standard C functions, but in addition to executable code they may also contain variable bindings to automatic (stack) or managed (heap) memory. A block can therefore maintain a set of state (data) that it can use to impact behavior when executed.
Blocks Programming Topics, Apple

语法

return_type (^name)(parameters) { ... }

截获变量

只读变量

block可以截获在声明block语法的函数作用域内的局部变量
实质：将在block语法内使用到的变量，赋值给block结构体内的相同类型变量

在block语法内，可以对截获的对象调用OC方法，但不可以对其赋值

读写变量

  • static var
  • static global var
  • global var
  • __ block 存储域类说明符 var

实质

Objective-C的对象，C的结构体

存储域

Block的种类

种类 所属类 存储域 生成 说明
栈块 _NSConcreteStackBlock 栈区 Stack 函数内 默认的block类型； 编译器会给每个块分配好栈内存，当离开了相应的作用域之后，编译器有可能把分配给块的内存覆写掉。
堆块 _NSConcreteMallocBlock 堆区 Heap 从栈复制到堆 带引用计数的块对象
全局块 _NSConcreteGlobalBlock 数据区域 .data区 在记述广域变量的地方声明的block； 不截获自动变量的blcok； 全局块不能使用自动变量 ，所以不会截获自动变量； 不随运行时变化而变化； 块所使用的整个内存区域，在编译期已经完全确定了。
